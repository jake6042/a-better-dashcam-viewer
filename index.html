About This Program

This desktop application exports Tesla dashcam footage with real-time overlay of vehicle telemetry data (speed, autopilot status, GPS coordinates, blinkers, etc.) embedded in the video stream.

To use this tool, your dashcam footage must be from a HW3+ vehicle running Tesla Software version 2025.44.25 or higher.

Privacy: Everything runs entirely on your computer. No data is uploaded to any server. Your videos and metadata never leave your device.

Built using Tesla's open-source utilities from github.com/teslamotors/dashcam

Not affiliated with Tesla, Inc. Donations support development of this desktop application only.
